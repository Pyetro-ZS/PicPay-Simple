# PicPay-Simple API

API REST em Spring Boot que simula transações entre usuários inspirada no desafio backend do PicPay.

## Tecnologias
Java • Spring Boot • JPA • Maven • MySQL/H2

## Funcionalidades
- Criar e listar usuários
- Simular transações com validação de saldo
- Tratamento de erros com mensagens claras
